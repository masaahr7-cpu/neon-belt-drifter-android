Neon Belt Drifter Android build guide

GitHub repository name: neon-belt-drifter-android
Package name: com.neonbeltdrifter.app

Fastest handoff:
1. In the game, open "Download full source + workflow".
2. If GitHub says too many files, download neon-belt-drifter-github-web-upload.zip.
3. Extract it.
4. Upload all extracted files to GitHub repo neon-belt-drifter-android.
5. Open GitHub Actions and run "Android APK/AAB Build".

1. Push this project to GitHub.
2. Open GitHub Actions.
3. Run "Android APK/AAB Build".
4. Download "Neon-Belt-Drifter-debug-apk" for phone testing.
5. Add signing secrets, run the workflow again, and download "Neon-Belt-Drifter-release-aab-signed" for Play Store upload.

Local output paths:
android/app/build/outputs/apk/debug/app-debug.apk
android/app/build/outputs/bundle/release/app-release.aab

Full guide is in docs/ANDROID_RELEASE_GUIDE.md in the repository.

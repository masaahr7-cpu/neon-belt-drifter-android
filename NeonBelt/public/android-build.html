If GitHub says “upload fewer than 100 files”, use neon-belt-drifter-github-web-upload.zip. It has fewer files and the workflow creates the Android project automatically.

Upload the ZIP contents to a GitHub repo named neon-belt-drifter-android. It includes the game source, Play Store docs, store art, privacy policy, and GitHub Action workflow.

After you push this project to GitHub, the workflow creates downloadable artifacts named:

  • Neon-Belt-Drifter-debug-apk for phone testing
  • Neon-Belt-Drifter-release-aab-signed for Google Play when signing secrets are configured

Replace YOUR_USERNAME in the workflow link after you create your GitHub repo. Then open the Actions run and download these artifacts:

  • Neon-Belt-Drifter-debug-apk
  • Neon-Belt-Drifter-release-aab-signed

Local output paths after running the commands:

android/app/build/outputs/apk/debug/app-debug.apk

android/app/build/outputs/bundle/release/app-release.aab

Workflow file content — copy this into .github/workflows/android-build.yml if download is blocked:

name: Android APK/AAB Build

on:
  workflow_dispatch:
  push:
    branches: [main]

jobs:
  build-android:
    name: Build Android artifacts
    runs-on: ubuntu-latest
    env:
      HAS_ANDROID_SIGNING: ${{ secrets.ANDROID_KEYSTORE_BASE64 != '' }}

    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: Set up Node
        uses: actions/setup-node@v4
        with:
          node-version: 20

      - name: Set up Java
        uses: actions/setup-java@v4
        with:
          distribution: temurin
          java-version: 21

      - name: Install dependencies
        run: npm install --include=dev

      - name: Build web app and sync Android
        run: npm run android:sync

      - name: Build debug APK
        run: cd android && ./gradlew assembleDebug

      - name: Build unsigned release AAB
        run: cd android && ./gradlew bundleRelease

      - name: Upload debug APK
        uses: actions/upload-artifact@v4
        with:
          name: Neon-Belt-Drifter-debug-apk
          path: android/app/build/outputs/apk/debug/*.apk

      - name: Upload unsigned release AAB
        uses: actions/upload-artifact@v4
        with:
          name: Neon-Belt-Drifter-release-aab-unsigned
          path: android/app/build/outputs/bundle/release/*.aab

      - name: Decode signing keystore
        if: env.HAS_ANDROID_SIGNING == 'true'
        run: |
          echo "${{ secrets.ANDROID_KEYSTORE_BASE64 }}" | base64 --decode > android/app/neon-belt-drifter-upload.jks

      - name: Sign release AAB
        if: env.HAS_ANDROID_SIGNING == 'true'
        run: |
          jarsigner \
            -keystore android/app/neon-belt-drifter-upload.jks \
            -storepass "${{ secrets.ANDROID_KEYSTORE_PASSWORD }}" \
            -keypass "${{ secrets.ANDROID_KEY_PASSWORD }}" \
            android/app/build/outputs/bundle/release/app-release.aab \
            "${{ secrets.ANDROID_KEY_ALIAS }}"

      - name: Upload signed release AAB
        if: env.HAS_ANDROID_SIGNING == 'true'
        uses: actions/upload-artifact@v4
        with:
          name: Neon-Belt-Drifter-release-aab-signed
          path: android/app/build/outputs/bundle/release/*.aab
